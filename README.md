# Vasudev Studio Hub 🎨

An interactive laboratory for high-performance visual asset creation. This project allows designers and developers to generate stunning mesh gradients, export design tokens, and download 4K assets.

## 🚀 Features

- **Mesh Engine**: Dynamic, fluid mesh gradients with physics-based motion.
- **Chromatic Studio**: Real-time color palette shuffling and preset management.
- **Motion Controls**: Adjust density and fluidity of the visual engine.
- **Production Export**: Download assets in various aspect ratios (16:9, 9:16, 1:1) and qualities (up to 4K).
- **Design Tokens**: Direct access to CSS variables for seamless implementation.

## 🛠️ Modules

- **Mesh Studio**: Current active module.
- *Coming Soon*: Blob Studio & Pattern Studio.
